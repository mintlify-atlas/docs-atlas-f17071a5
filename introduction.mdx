<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-semibold text-gray-900 dark:text-white mb-3">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get Scrapyd running in minutes and schedule your first spider
  </p>

  <Steps>
    <Step title="Install Scrapyd">
      Install Scrapyd using pip:
      
      ```bash
      pip install scrapyd
      ```
      
      Scrapyd requires Python 3.10 or higher and works on Linux, macOS, and Windows.
    </Step>

    <Step title="Start the daemon">
      Launch the Scrapyd service:
      
      ```bash
      scrapyd
      ```
      
      By default, Scrapyd listens on `http://localhost:6800`. You should see output indicating the server has started:
      
      <Accordion title="Expected output">
        ```
        2024-03-05T10:15:23-0500 [-] Scrapyd web console available at http://127.0.0.1:6800/
        2024-03-05T10:15:23-0500 [-] Loaded.
        ```
      </Accordion>
    </Step>

    <Step title="Deploy your project">
      Package and deploy your Scrapy project using scrapyd-client:
      
      ```bash
      pip install scrapyd-client
      scrapyd-deploy
      ```
      
      This builds a Python egg from your project and uploads it to Scrapyd via the `addversion.json` API endpoint.
    </Step>

    <Step title="Schedule a spider">
      Schedule a spider to run using the HTTP API:
      
      ```bash
      curl http://localhost:6800/schedule.json -d project=myproject -d spider=myspider
      ```
      
      <Accordion title="Response">
        ```json
        {
          "status": "ok",
          "jobid": "6487ec79947edab326d6db28a2d86511e8247444",
          "node_name": "node1"
        }
        ```
      </Accordion>
      
      Your spider is now running! View logs and monitor progress at `http://localhost:6800/`.
    </Step>
  </Steps>
</div>
